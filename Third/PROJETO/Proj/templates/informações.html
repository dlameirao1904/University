Proposto pelo docente José Carlos Ramalho, este pequeno projeto consiste numa pequena aplicação web para consulta e pesquisa de informação. Informação esta cuja consiste na criação de um modelo normalizado para as bases jurídicas atualmente disponíveis e também armazenar essa informação numa base de dados em MongoDB

A informação anteriormente falada consiste num conjunto de ficheiros json que representam o dataset de um tribunal específico.

Mas o que são esses datasets?

São, na sua forma estrutural, uma lista de dicionários, estrutura esta simples de manusear, facilitando a vida de quem trabalha de forma integra com este tipo de dados.

Cada dicionário, contém um conjunto de chaves e os seus respetivos valores, sendo estes, respetivamente, os campos de cada acórdao e a correspondente resposta a este mesmo.

Como podemos ver na figura, são estruturas, que simplificam todo o trabalho, de forma organizada.
